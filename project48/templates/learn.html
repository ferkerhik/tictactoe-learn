Register

การจะเข้าไปดู Profile และประวัติการเล่นของเรานั้นเราจะต้องทำการ Login ก่อน ซึ่งถ้าเรายังไม่มี ID สามารถ Register ได้ที่ปุ่มนี้

Sign Up

1.ใส่user nameที่ตัวเองคิด

2.ใส่ Email ที่ต้องการใช้ในการสมัคร

3.ตั้ง Password

4.กฎที่ใช้ในการตั้ง Password

5.ใส่ Password อีกครั้ง

6.กดยืนยันที่ปุ่มนี้

Login

หลังจากทำการregisterไปแล้ว สามารถกดloginได้ที่ปุ่มนี้

Sign In

1.ใส่user nameที่เราใช้ตอน sign up

2.ใส่ password ที่เราใส่ไปตอน sign up

3.กดยืนยันที่ปุ่มนี้
